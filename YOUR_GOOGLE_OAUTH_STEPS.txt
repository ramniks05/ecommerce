========================================
TROUBLESHOOTING
========================================

Problem: "Invalid API key"
Solution: Check .env.local file exists and has correct values

Problem: "redirect_uri_mismatch"
Solution: Make sure callback URL in Google Console matches Supabase exactly

Problem: Server not loading env variables
Solution: Restart server with Ctrl+C then npm run dev

Problem: Google login works but user not created
Solution: Check if database-schema.sql was run successfully in Supabase
